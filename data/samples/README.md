# Sample Files

Example inputs and expected outputs demonstrating the different input methods supported by AI Flashcard Generator.

## Files

| File | Description |
|------|-------------|
| `ocr-handwritten-notes.md` | Simulated OCR output from a photo of handwritten UX Design notes, with expected flashcard output |

## Purpose

These samples serve as:
- **Test data** for developing and validating the flashcard generation pipeline
- **Documentation** showing users what each input method looks like in practice
- **Reference** for expected flashcard quality and format
